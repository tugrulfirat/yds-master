# YDS Master — image generation prompts

One prompt per image. After generating, save each file with the EXACT name
shown (that's how the app finds it — see ASSETS.md), drop it into
`YDSMaster/Assets.xcassets` in Xcode, and it appears in the game automatically.

Golden rules for every generation:
- ONE object only, centered, nothing else in frame
- NO words, letters, numbers, labels, or captions anywhere in the image
- Transparent background (if your generator can't, use a plain solid #00FF00
  green background and I'll remove it)
- No ground shadow
